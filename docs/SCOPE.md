Project Scope (Frozen)

IN SCOPE:
- Client-side file encryption (AES-GCM)
- User key pair generation (Web Crypto API)
- Encrypted file upload
- Permission grant via key re-wrapping
- Time-bound access tokens
- Access revocation via wrapped-key deletion
- Demo external application

OUT OF SCOPE:
- Blockchain
- Mobile app
- Decentralized identity
- Full production-grade Proxy Re-Encryption
- Enterprise authentication
